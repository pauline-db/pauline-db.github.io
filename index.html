FlyForce - 3-DOF Force Sensor

Date: Spring 2025

Author: Pauline De Baets, Stanislas Dupanloup, Emmard Frangopoulos, Noé Ventura, Tim Vial

FlyForce is a mechanical design project focused on the development of a three-degree-of-freedom force sensor based on flexure mechanisms. The sensor was designed to measure the contact force between a spherical stylus tip and a part in all directions, while minimizing sensitivity to gravity and base accelerations.

The mechanism uses elastic deformation, compliant guides, force and moment balancing, and capacitive displacement sensing to estimate the applied force. The final concept includes a flexible guiding system allowing two rotations and one translation, as well as balancing mechanisms designed to improve reliability and measurement stability.

Through analytical dimensioning and CAD-based design, the project achieved stiffness values close to the target of 5 N/mm in all three directions, with force resolutions below the required 1.5 mN. This project strengthened my experience in compliant mechanism design, precision engineering, CAD, mechanical modelling, and design iteration under industrial constraints.

Pac-Man on ATmega128L

View this project on GitHub

Date: Spring 2025

Authors: Pauline De Baets, Stanislas Dupanloup

A Pac-Man-inspired game developed in AVR Assembly for the ATmega128L microcontroller board.

The game runs on an 8×8 WS2812B LED matrix, uses an LCD 2×16 to display the menu and score, a PS/2 keyboard for menu navigation, push buttons for player movement, and the internal EEPROM to store high scores.

The video on the right shows a demonstration of the game, where players move through the maze, collect bonuses, avoid or eat enemies, and reach the green exit. After completing the first level, the game automatically starts a second, harder level. When the second level is completed, a green check mark is displayed on the LED matrix. The video only shows the led matrix forthe first part of the game, where the player (yellow) navigates the maze, collects a bonus (dark blue) to eat the enemy (red). He then proceeds to exit the first level and starts the second.

Radar Gesture Recognition

View this project on GitHub

Date: October 2025

Author: Pauline De Baets

Radar-based hand gesture recognition using micro-Doppler signatures and machine learning using a dataset of four different hand gestures: click, pinch, swipe, and wave from 6 different participants. This project compares Chebyshev moments and micro-Doppler envelope features for classifying four hand gestures.

Feature Extraction Method Classifier Best Accuracy
Chebyshev Moments k-NN 88.65%
Chebyshev Moments 1D CNN 86.31%
Chebyshev Moments Quadratic SVM 83.05%
Micro-Doppler Envelopes k-NN 67.0%
Micro-Doppler Envelopes Linear SVM 65.1%
Micro-Doppler Envelopes Logistic Regression 65.1%
